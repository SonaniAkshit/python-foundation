# Practical Problem Definitions

1. Create a program that stores a student's name, age, and percentage using variables and prints the complete student profile.

2. Build a calculator that takes two numbers as input from the user and performs all arithmetic operators like `+`, `-`, `*`, `/`, `%`.

3. Write a program that accepts a user's birth year as input and calculates their current age using variables and operators.

4. Develop a system that converts temperature from Celsius to Fahrenheit using type conversion and arithmetic operators.

5. Create a billing program that takes product price and quantity as input and prints the total bill amount.

6. Write a program that checks whether a number entered by the user is even or odd using comparison and modulus operators.

7. Build a program that takes a user's first name and last name separately and combines them into a full name using string variables.

8. Create a marks analysis program that accepts marks as string input, converts them into integers, and calculates the average score.

9. Develop a salary calculator that takes basic salary as input and calculates bonus and final salary using assignment and arithmetic operators.

10. Write a program that accepts two numbers as strings from the user, converts them into integers using type conversion, and prints their sum, difference, and multiplication result.
