# đź§  Command Mode vs Edit Mode

This confuses almost everyone initially.

## Edit Mode

* Cursor blinking inside cell
* You are typing code

## Command Mode

* Cell border highlighted
* Shortcuts like `A`, `B`, `D`, `M` work

Switch:

* `Enter` â†’ Edit mode
* `Esc` â†’ Command mode
