# ✅ Practice Tasks for Loop Mastery

Complete these hands-on tasks to solidify your understanding of `for` loops, `while` loops, `range`, nested loops, and logical problems with loops.  
Mark each task as `[x]` when finished.

---

## 🔘 For Loop

- [ ] **Task 1: Sum of squares**  
  Write a `for` loop that computes the sum of squares of numbers from 1 to `n` (user input). Example: `n=4` → 1²+2²+3²+4² = 30.

- [ ] **Task 2: Character frequency counter**  
  Given a string, use a `for` loop to count how many times each character appears. Store results in a dictionary. Ignore case (`'A'` and `'a'` count together).

- [ ] **Task 3: Reverse a list without slicing**  
  Use a `for` loop to reverse a list in place (modify the original list). Example: `[1,2,3,4]` → `[4,3,2,1]`.

- [ ] **Task 4: Flatten a mixed list**  
  Given `[1, [2, 3], 4, [5, [6, 7]]]`, write a `for` loop (and maybe a helper) to flatten it completely into `[1,2,3,4,5,6,7]`. (This requires conditional logic inside the loop.)

- [ ] **Task 5: Enumerate without `enumerate()`**  
  Recreate the behaviour of `enumerate()` using a manual counter inside a `for` loop. Iterate over a list and print both index and value.

## 🔘 While Loop

- [ ] **Task 1: Guess the number (with limited attempts)**  
  Generate a random secret number (1–100). The user has 7 attempts. Use a `while` loop to keep asking. After each guess, tell “too high” or “too low”. If guessed correctly, print “You win!” and break. If attempts exhausted, print “You lose! The number was X”.

- [ ] **Task 2: Collatz sequence length**  
  Write a function that, given a positive integer `n`, uses a `while` loop to compute the number of steps to reach 1 (Collatz conjecture: if even → n//2, if odd → 3n+1). Return the number of steps. Example: `collatz_steps(6)` → 8 (6→3→10→5→16→8→4→2→1).

- [ ] **Task 3: Sum of digits until single digit**  
  Keep summing the digits of a number until the result is a single digit. Use a `while` loop. Example: `9875` → 9+8+7+5=29 → 2+9=11 → 1+1=2. Return 2.

- [ ] **Task 4: Validate input with a sentinel**  
  Ask the user to enter positive integers. Stop when they enter `0` or a negative number. Print the count of numbers entered and their average (ignoring the sentinel).

- [ ] **Task 5: Simulate a countdown with user interrupt**  
  Write a countdown from 10 to 0. At each second (use `time.sleep(1)`), print the remaining time. If the user presses Ctrl+C (catch `KeyboardInterrupt`), ask “Really quit? (y/n)`. If `y`, break; else continue countdown.

## 🔘 Range Function

- [ ] **Task 1: Reverse range without `reversed()`**  
  Use `range` with negative step to print numbers from 20 down to 1 (inclusive).

- [ ] **Task 2: List of even squares**  
  Create a list of squares of all even numbers from 2 to 20 (inclusive) using `range` and a list comprehension (or a `for` loop with `range`). Result: `[4, 16, 36, 64, 100, 144, 196, 256, 324, 400]`.

- [ ] **Task 3: Efficient membership test**  
  Check if the number `1234567` is in `range(0, 10**7, 7)` without converting to a list. Print the result and explain why it’s fast.

- [ ] **Task 4: Generate arithmetic progression**  
  Given `start = 5`, `stop = 50`, `step = 4`, use `range` to generate the sequence. Then compute the sum of that sequence using a formula (not a loop). Verify with a loop.

- [ ] **Task 5: Custom step that is not an integer**  
  Explain why `range(0, 10, 0.5)` fails. Then write a function `frange(start, stop, step)` that yields floating‑point numbers (like a generator) using a `while` loop. Example: `list(frange(0, 1, 0.25))` → `[0, 0.25, 0.5, 0.75]`.

## 🔘 Nested Loops

- [ ] **Task 1: Multiplication table up to 12×12**  
  Print a nicely formatted multiplication table (12 rows × 12 columns). Each number should be right‑aligned in 4 spaces.

- [ ] **Task 2: Pascal’s triangle (up to 6 rows)**  
  Use nested loops to print Pascal’s triangle.  
  Row `i` contains binomial coefficients `C(i, j)` for `j=0..i`. You may compute each value by adding above two, or use the combination formula.

- [ ] **Task 3: Unique pairs from two lists**  
  Given `names = ["Alice", "Bob", "Charlie"]` and `tasks = ["Clean", "Cook", "Write"]`, print all possible (name, task) assignments. Then modify to ensure each person gets a unique task (no repeats) – that requires additional logic.

- [ ] **Task 4: Matrix transpose**  
  Write a function that takes a 2D list (matrix) and returns its transpose using nested loops (no `zip`). Example: `[[1,2,3],[4,5,6]]` → `[[1,4],[2,5],[3,6]]`.

- [ ] **Task 5: Find the maximum sum sub‑rectangle**  
  Given a 2D matrix of integers (e.g., 4×4), use four nested loops (top, bottom, left, right) to find the sub‑rectangle with the largest sum. This is the naive O(n⁴) solution. (For a small matrix, it’s fine.)

## 🔘 Loop Logical Problems

- [ ] **Task 1: Find the missing number**  
  You are given a list containing `n-1` numbers from 1 to `n` (one number missing). Use a loop to find the missing number without using extra space (except a few variables). Example: `[1,2,4,5]` with `n=5` → missing 3.

- [ ] **Task 2: Remove duplicates in place**  
  Given a sorted list, use a loop to remove duplicates **in place** (without creating a new list). Return the new length. Example: `[1,1,2,2,2,3,4,4,5]` → `[1,2,3,4,5]` and length 5.

- [ ] **Task 3: Leader elements**  
  An element is a leader if it is greater than all elements to its right. Write a loop (from right to left) to collect all leaders. Example: `[16, 17, 4, 3, 5, 2]` → leaders `[17,5,2]`.

- [ ] **Task 4: Two‑sum with sorted array (two pointers)**  
  Given a sorted list and a target, use two pointers (one at start, one at end) and a `while` loop to find if any pair sums to the target. Return the pair indices. Do not use nested loops.

- [ ] **Task 5: Simulate a dice game**  
  Two players roll a die alternately. The first to reach a cumulative score of 50 wins. Each roll adds the result to that player’s score. Use a `while` loop, `random.randint(1,6)`, and print whose turn it is and the score after each roll. At the end, announce the winner.

- [ ] **Task 6: Binary search implementation**  
  Implement binary search using a `while` loop. Given a sorted list and a target, return the index or `-1` if not found. Test with `[2,5,8,12,16,23,38,56,72,91]` and target `23`.

- [ ] **Task 7: Rotate a list by k positions**  
  Write a function that rotates a list to the right by `k` steps using a loop and without using slicing (simulate element‑by‑element shift). Example: `[1,2,3,4,5]`, k=2 → `[4,5,1,2,3]`.

## 📌 Tips for Success

- Run each task with different test inputs, including edge cases (empty list, single element, large numbers).
- For time‑consuming tasks (like large nested loops), start with small test sizes and increase gradually.
- After solving, look for more Pythonic ways using built‑ins – but keep your loop version to understand the logic.

Happy looping! 🐍
