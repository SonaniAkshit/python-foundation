# ML & Data Science Connection

Collections are everywhere in Data Science:

| Collection           | ML/Data Science Usage |
| -------------------- | --------------------- |
| List                 | Raw data storage      |
| Tuple                | Immutable records     |
| Set                  | Unique categories     |
| Dictionary           | Feature mappings      |
| Nested Dictionary    | JSON APIs             |
| List of Dictionaries | Dataset records       |
| Dictionary of Lists  | DataFrame creation    |

Example:

```python
data = {
    "name": ["Akshit", "Rahul"],
    "age": [21, 22]
}
```

This directly becomes a Pandas DataFrame.
